📋 À propos du projet

Dionysos Survie Vania Wiki est une application web servant de base de connaissances interactive pour le futur serveur de survie de notre communauté Discord "Dionysos".

Le serveur n'étant pas encore sorti, ce projet est actuellement en pleine construction pour préparer le lancement. Il est conçu pour centraliser sur plusieurs pages toutes les informations nécessaires aux futurs joueurs : règles, histoire (lore), présentation des villes et guides de démarrage.

Ce projet met en œuvre une architecture frontend moderne avec React, offrant une navigation instantanée et fluide entre les différentes pages du wiki sans aucun rechargement.

🔗 URL du dépôt : https://github.com/DavidChloe/dionysos-survie-vanilla-wiki

✨ Fonctionnalités Clés

   📖 Base de Connaissances : Consultation de l'histoire du serveur, des événements et des systèmes de points.
    📜 Règles et Guides : Espace dédié aux règles de la survie Vania et explications pour bien débuter.
    ⚡ Navigation Ultra-Rapide : Changement de page instantané grâce au routage côté client.
    📱 Design Responsive : Interface adaptée pour être consultée facilement sur mobile (idéal pour les joueurs sur Discord).

🛠️ Stack Technique

    Frontend : React 18 (Hooks personnalisés, Context API).
    Routage : React Router v6 (Pour la gestion des multiples pages).
    Styling : Tailwind CSS.

🚀 Installation et Configuration

Pour exécuter le Wiki Dionysos localement sur votre machine :

1. Cloner le dépôt

```bash
git clone [https://github.com/DavidChloe/dionysos-survie-vania-wiki.git](https://github.com/DavidChloe/dionysos-survie-vania-wiki.git)
cd dionysos-survie-vania-wiki
```

2. Installer les dépendances

```bash
npm install
cd dionysos-survie-vania-wiki
```

3. Lancer l'application

```bash
npm start
```

L'application sera accessible sur http://localhost:3000.
